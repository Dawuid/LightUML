Introduction to LightUML

LightUML is a lightweight alternative for generating UML class diagrams in Eclipse. A class diagram can be generated from a Java project or a Java package (which may contain other packages).

LightUML uses UmlGraph doclet (by Diomidis Spinellis) to generate a .dot file out of a Java project or package. After that (AT&T) GraphViz 'dot' tool is used to convert the .dot file to graphics form of choice. And lastly, the graphics file is added to the Java project.

As a prerequisite, you need to have GraphViz installed. You also need to get a UmlGraph jar-package.

LightUML also aims to support UMLGraph sequence diagrams. This part is still under construction but a prerequisite for generating sequence diagrams is the plotutils package.

See LightUML homepage for instructions and up to date info.
